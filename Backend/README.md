# Task Management System

A full stack Task Management System developed using React JS for frontend and Spring Boot for backend.
This project includes role-based authentication, validation, relationship, REST APIs, database integration, and task management features.
Here users can create account.There are two roles:Admin and Member.Admin has permission to create and assign tasks.Member can only view assigned tasks.
After login as Admin, dashboard is opened.Admin can:Create tasksManage workflowView assigned tasksThen login as MEMBER.Member dashboard has limited access.
This implements role-based authorization.This is the H2 database console.All user and task data is stored here.
Here I am testing APIs using Postman.Backend APIs are connected successfully with frontent.
This project helped me understand full stack development, API integration, authentication, database connectivity, and frontend-backend communication.

## 🚀 Features
- User Signup and Login- Role Based Authentication- Admin Dashboard- Member Dashboard- Task Creation
- Task Assignment- REST API Integration- H2 Database Integration further updated to sql- Frontend and Backend Communication- Responsive UI
